# Caramak - Smart Transport Tracking & Passenger Awareness System

A modern React Native mobile app for real-time tracking of buses and keke (tricycles) with passenger availability awareness.

## ✨ Features

- Beautiful onboarding flow
- User authentication screen (Email + Google/Apple ready)
- Simulated live tracking map (Abuja)
- Real-time ride booking with nearby keke and buses
- Clean, modern UI matching the original design

## 🛠 Tech Stack

- **Frontend**: React Native + Expo
- **Navigation**: React Navigation
- **UI**: Linear Gradient + Custom styling
- **Map**: react-native-maps (simulated for demo)

## 📱 Screens

1. **Welcome Screen** - App introduction with keke illustration
2. **Create Account** - Email/password + social login placeholders
3. **Onboarding** - Explains real-time tracking feature
4. **Home Screen** - Live map + Available rides with booking

## 🚀 How to Run

```bash
# Install dependencies
npm install

# Start the development server
npx expo start
