Caramak
Smart campus shuttle tracking system 
Overview
Caramak is a smart transport tracking system designed to help students locate campus buses and keke in real time.
it reduces waiting time and improves transport efficiency using GPS tracking and IOT concepts.

Problem
Students often wait for long periods without knowing where campus buses or keke are. This leads to missed lectures 
and fustration.

Solution
Caramak provides:
Real-time vehicle tracking
Estimated arrival time
Bus/keke availability status (Full/Not Full)

Technologies Used
Frontend: React
Backend: (Node.js)
Database: Mongodb
Hardware: ESP32 + IR sensors
GPS tracking
